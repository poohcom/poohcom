Detailed Description

JEUS에서 Web HMI의 태그 데이터를 수집하고, 이를 요청이 있는 웹 클라이언트에 전달해주기 위한 서비스 메인 클래스

수정일시: 2014. 10. 06 수정자: BIGGI 수정내용:

  1. 클라이언트 요청에서 WAS노드 정보 요청에 대응하는 서비스가 생기면서, WAS 노드 정보를 반환하는 getLocalNode() 메서드가 신규로 만들어 짐.
