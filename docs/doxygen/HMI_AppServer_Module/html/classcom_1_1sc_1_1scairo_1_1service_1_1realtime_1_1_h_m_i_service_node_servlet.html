Detailed Description

HMIServiceNodeServlet.class

HMI 웹클라이언트의 요청에 의해 해당 WAS가 서비스하는 노드명을 반환하는 서블릿. 기존에는 애플릿 JNLP 파일 내에 "hmi.service.node" 속성으로 정의되어 사용되는 것을, JNLP Parser가 한글을 예외처리하면서 부터(7u65 이후) 이 서비스를 이용하여 조회해 가는 것으로 변경함.
