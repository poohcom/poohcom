Detailed Description

ControlDBLogger에 의해 처리되는 제어 이력 쿼리 중 연결 실패의 원인으로 처리되지 못한 내용들을 저장하고 재처리하기 위한 클래스. 쿼리 자체에 문제가 있는 경우에는 FAILED_QUERY 로그에 저장된다.

먼저 등록된 재처리 쿼리가 항상 먼저 시작되어야 하며, 재처리시 실패할 경우에도 다음 실행시 가장 먼저 실행될 쿼리로 처리되어야 한다.
