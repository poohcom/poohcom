Detailed Description

외부 시스템이 XML 형식으로 태그 데이터를 요청할 경우에 처리하는 Servlet 요청된 태그 데이터를 확인하여 반환한다. 요청 형태는 POST형식으로 데이터를 "tags=tagid1;tagid2;..." 붙여서 요청하면 요청된 태그 중 서버가 관리하는 태그에 대한 현재 데이터만을 반환한다.
