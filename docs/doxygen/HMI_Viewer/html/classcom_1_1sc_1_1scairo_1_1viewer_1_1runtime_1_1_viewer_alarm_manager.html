Detailed Description

뷰어 내에서 태그사전의 경보객체를 활성화시켜 주기 위한 관리 클래스 뷰어 내에서 경보 객체들은 AlarmStateCondition에 의해 동작하는 화면 효과들을 운영하기 위해 활성화되며, 실제 데이터통합서버에서 생성되는 경보 데이터는 별도로 유지된다.
