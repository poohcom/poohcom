◆ updateData()

void com.sc.scairo.viewer.runtime.AlarmPopupDialogManager.updateData ( AlarmData  data)

기존 경보 데이터의 상태가 변경되 경우 처리하는 메서드.

  1. 우선 해당 경보 데이터가 리스트 또는 팝업창에 있지 않으면 그대로 반환된다.
  2. 현재 경보 데이터가 필터를 통과하면 리스트 또는 팝업창의 내용을 변경하여 표시한다.
  3. 필터를 통과하지 못하면 리스트에서 삭제하거나 팝업창을 닫는다.(AutoClosing시)
