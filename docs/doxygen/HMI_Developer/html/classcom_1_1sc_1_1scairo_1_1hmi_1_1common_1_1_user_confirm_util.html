Detailed Description

각종 사용자 결정에 대한 화면을 출력하고, 그에 따른 사용자 선택을 반환해주는 공통 클래스 반환값은 JOptionPane의 YES_, NO_, CANCEL_OPTION 상수를 이용한다.

프로젝트 정보의 변경 유무를 확인하여 사용자에게 저장 여부를 질의하여 처리하고, 이후 과정(호출하는 쪽에서 진행되는)을 계속 수행할지에 대한 결과를 반환하는 메서드.

프로젝트 변경 상태가 없을 경우에는 저장하지 않고, 과정 수행을 진행하도록 true 반환 프로제트 변경이 있을 경우 사용자에게 저장 여부를 문의하여 사용자가 YES 선택시 프로젝트 저장후, 과정 수행을 진행하도록 true 반환 NO 선택시 프로젝트 저장하지 않고, 과정 수행을 진행하도록 true 반환 CANCEL 선택시 프로젝트 저장하지 않고, 과정 수행을 진행하지 않도록 false 반환
