Detailed Description

화면의 모든 내용을 일반 텍스트 문서(CSV, XML)로 저장하는 화면.

현재 프로젝트에서 작성한 모든 화면 목록이 나타나며, 일반 텍스트로 저장하려는 화면을 선택한 뒤 저장 형식을 선택하는 형식으로 구성된다. 저장 형식은 CSV, XML 두가지를 지원하며, 현재는 CSV 형식만 선택할 수 있다.

내보내기는 HMIDrawingImporter#exportToCSV(com.sc.scairo.core.uclair.draw.common.ViewList, File[], File) 메서드를 사용하며, HMIDrawingImporter 에 PropertyChangeListener 를 추가하여, 진행 상황 및 내보내기 결과를 확인한다.

수정일시: 2014. 3. 14 수정자: BIGGI 수정내용:

  1. XML관련 기능이 추가되면서 기존에 disable되어 있던 xml버튼을 활성화시킴.
  2. HMIDrawingImporter의 import/export 관련 메서드가 수정되어, 이에 대한 호출 코드를 변경함.
