Detailed Description

시스템 편집기에서 스크립트에 대한 검사를 수행할 때 사용할 CommonTagFunctionImpl의 상속 클래스 실제 태그에 대한 작업은 수행하지 않으며, 태그에 대한 내용이 존재하거나 수행할 수 있는 기능인지만을 검사할 수 있도록 CommonTagFunctionImpl의 구현된 내용들을 override한다.
