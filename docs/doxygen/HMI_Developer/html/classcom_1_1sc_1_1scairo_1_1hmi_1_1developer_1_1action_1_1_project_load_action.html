Detailed Description

수정일시: 2014. 10. 22 수정자: BIGGI 수정내용:

  1. 프로젝트 암호 추가에 따라 프로젝트 열기시 암호가 지정된 프로젝트의 경우 암호를 검사하도록 하였고, 암호 실패시 프로젝트 로딩이 되지 않도록 함 actionPerformed(ActionEvent) 수정됨
