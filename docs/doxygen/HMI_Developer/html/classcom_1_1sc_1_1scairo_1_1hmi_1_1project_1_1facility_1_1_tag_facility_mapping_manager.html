Detailed Description

수정일시: 2014. 07. 22 수정자: BIGGI 수정내용:

  1. 시설물태그매핑에시 상위 그룹에 시설코드를 넣고 하위 자동 생성을 선택하였을 때, 다음 메서드가 실행되면서 하위 태그들에 자동으로 시설물코드 매핑이 이루어질 때, 하위 그룹태그에는 데이터코드가 입력되지 않는 코드를 추가함. facilityChangeChild(GroupTag, String)
  2. 시설물 코드 및 데이터 코드 자리수 확인시 기존의 문자열 길이(byte단위가 아닌 JAVA문자열 단위)로 검사하면 DBMS의 데이터 길이와 차이가 발생한다. 이를 맞추기 위해 KECO SIMS에서는 DB가 UTF-8로 정의되어 있으므로 이를 적용하도록 다음 메서드를 수정함 saveFacility(Tag)

yongwoo Park
