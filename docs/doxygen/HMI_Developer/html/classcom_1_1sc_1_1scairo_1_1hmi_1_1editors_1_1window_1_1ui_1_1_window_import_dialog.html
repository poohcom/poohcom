Detailed Description

일반 텍스트(CSV 또는 XML)로 저장된 파일로 부터 화면을 생성한다.

단일 파일 혹은 폴더를 지정할 수 있으며, 현재 프로젝트에 새로운 화면 파일로 생성할 것인지 현재 편집 중인 화면에 표시할 것인지 선택하여 화면을 생성한다. 파일 읽기 형식은 CSV, XML 두가지를 지원하며, 현재는 CSV 형식만 선택할 수 있다.

가져오기는 단일 파일 지정시HMIDrawingImporter#importFromCSV(File, HMIDesignView), 폴더 지정시 HMIDrawingImporter#importFromCSV(File, File, ViewList) 메서드를 사용하며, HMIDrawingImporter 에 PropertyChangeListener 를 추가하여, 진행 상황 및 가져오기 결과를 확인한다.

수정일시: 2014. 3. 14 수정자: BIGGI 수정내용:

  1. XML관련 기능이 추가되면서 기존에 disable되어 있던 xml버튼을 활성화시킴.
  2. HMIDrawingImporter의 import/export 관련 메서드가 수정되어, 이에 대한 호출 코드를 변경함.
