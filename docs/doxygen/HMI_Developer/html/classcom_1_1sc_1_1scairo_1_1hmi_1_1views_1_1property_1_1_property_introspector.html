Detailed Description

JavaBean 클래스의 BeanInfo 클래스로 부터 PropertyView에서 편집 가능한 com.jidesoft.grid.Property 클래스로 변환시켜주기 위한 클래스.

com.jidesoft.introspector.BeanIntrospector 가 BeanInfo로 부터 만들어주기 위한 유틸리티 클래스이나, 기본 형식의 데이터가 아닌 경우 표시 및 편집할 수 없으므로 이를 지원하기 위해 생성된 클래스이다.
