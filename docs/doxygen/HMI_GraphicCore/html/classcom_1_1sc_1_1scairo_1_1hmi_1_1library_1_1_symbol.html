Detailed Description

Symbol 라이브러리 내에서 등록된 하나의 객체 정보를 정의하는 클래스. 표현을 위한 데이터 및 편집기 상에서 객체를 라이브러리 내에서 표시 가능한 표시 컴포넌트를 제공한다.

수정일시: 2014. 3. 20 수정자: BIGGI 수정내용:

  1. defaultSvg, onSvg, faultSvg 는 사용되지 않는 속성이므로 관련 내용 삭제
  2. setSymbolData(Object)에서 TYPE이 SVG일 경우에는 cache svg를 로딩하기 위해 loadSvgDocument() 호출하도록 수정됨.
