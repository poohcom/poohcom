Detailed Description

SelectAreaTracker implements interactions with the background area of a Drawing.

The SelectAreaTracker handles one of the three states of the SelectionTool. It comes into action, when the user presses the mouse button over the background of a Drawing.

Author
Werner Randelshofer
Version
3.0 2006-02-15 Updated to handle multiple views.
1.0 2003-12-01 Derived from JHotDraw 5.4b1.
