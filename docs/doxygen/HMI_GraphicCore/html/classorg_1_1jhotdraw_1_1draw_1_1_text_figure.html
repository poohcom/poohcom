Detailed Description

A text figure.

A DrawingEditor should provide the TextTool to create a TextFigure.

Author
Werner Randelshofer
Version
2.0.2 2007-05-02 Made all instance variables protected instead of private.
2.0.1 2006-02-27 Draw UNDERLINE_LOW_ONE_PIXEL instead of UNDERLINE_ON.
2.0 2006-01-14 Changed to support double precison coordinates.
1.0 2003-12-01 Derived from JHotDraw 5.4b1.
