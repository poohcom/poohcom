Detailed Description

사용자의 마우스 끌기 이벤트에 의해 태그값을 변경시킨다. 이 동작은 사용자가 마우스를 드래그하는 동안에는 직접 태그의 값을 변경시키며, Release하는 동작에서 태그의 값을 제어한다.
