Detailed Description

그래픽 라이브러리 클래스. 라이브러리는 카테고리의 집합으로 구성된다. 각 카테고리는 표시 이름과 라이브러리 내의 유일 id (정수)로 구성된다.

수정일시: 2014. 3. 20 수정자: BIGGI 수정내용:

  1. initFromJar(GraphicLibrary)에서, Bean Symbol 구성 시에 Symbol에 입력되는 data를 Bean Class 객체에서 Bean ClassName(String)으로 변경(JRE호환성)
  2. 사용되지 않는 defaultSvgMap, onSvgMap, faultSvgMap 속성 및 관련 내용 삭제
  3. CATEGORY_LAST_ID 상수 삭제 (사용되지 않음)

TODO 라이브러리 구성 시에 현재는 등록명을 각 심볼의 키로 사용하므로써, 국제화시에 작업 내용이 문제가 발생되므로 각 Category 및 Symbol id를 국제화에 문제 없도록 변경하여 사용되도록 수정되어야 한다.
