Detailed Description

1.static으로 정의되어 있던 load 메서드 3개를 삭제함 2.HMIDrawingLoader를 이용하거나 View를 이용하여 로딩되도록 함. 주석처리되어 있던 DEFAULT_ID, defaultIndex 속성 삭제 처리함. 기타 주석 처리되어 있던 메서드도 같이 삭제 3.read(DOMInput), write(DOMOutput)의 경우 XML 처리이므로 별도 효과 파일(link)이 존재하지 않으므로 XML 파일 내에 같이 로딩/저장되도록 코드 추가됨.
