Detailed Description

프로젝트 내에서 사용된 SVG 라이브러리 심볼들의 저장 및 로딩을 담당하는 클래스. 화면 내에서 사용되는 SVG 심볼은 ID만 저장되고 실제 SVG 데이터는 이곳에 저장되어 관리된다.
