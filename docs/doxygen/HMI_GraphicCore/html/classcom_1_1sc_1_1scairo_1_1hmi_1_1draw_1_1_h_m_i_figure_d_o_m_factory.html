Detailed Description

HMIFigureDOMFactory JHotDraw의 In/Out 포맷 중에서 XML포맷을 사용할 경우에 각 객체들의 XML 태그를 정의하는 클래스.

수정일시: 2014. 3. 13 수정자: BIGGI 수정내용:

  1. HMIRectFigure에 대한 XML 태그명을 rect -> hmi_rect로 변경
  2. FillForm에 대한 XML 태그명을 추가함.
