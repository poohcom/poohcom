Detailed Description

EffectLoader 화면과 연관된 HMIDrawingEffects의 로딩 및 활성화/비활성화를 담당하는 유틸리티 클래스.

수정일자: 2014. 3. 20 수정자: BIGGI 수정내용:

  1. XML 화면을 운영할 경우에는 별도의 link파일이 존재하지 않고 XML내에 효과가 같이 저장된다. 이에 따라 별도 load/save를 하지 않고 HMIDrawing 내의 로딩된 HMIDrawingEffects를 이용하여 활성화/비활성화하는 다음 두 개의 메서드가 추가됨. activateDynamicLinkForXml(String, HMIRuntimeView, TagDictionary), deactivateDynamicLinkForXml(HMIDrawingEffects)

수정일자: 2014. 4. 3 수정자: BIGGI 수정내용:

  1. 메서드 내용 변경에 따라 다음 메서드 수정 해당 메서드 호출시 화면에 없는 객체들에 대한 효과리스가 넘어오므로 해당 ID 출력하도록 수정함 #activateDynamicLink(File, HMIRuntimeView, TagDictionary) #activateDynamicLink(String, HMIRuntimeView, TagDictionary) #activateDynamicLink(URL, String, HMIRuntimeView, TagDictionary) BIGGI 2014. 3. 20. : 오전 11:46:13
