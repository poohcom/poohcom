Detailed Description

화면 상에서 운영될 Bean객체들을 편집을 위하여 담는 Figure 클래스. JComponent를 상속한 JavaBean 객체를 운영하도록 구성되어져 있다.

  1. 3. 18 수정 1.readExternal/writeExternal() 1) bean 객체를 bean = in.readObject(), out.writeObject(bean)을 이용하여 처리하던 것을 bean을 HMIBean으로 Cast하여 ((HMIBean) bean).readExternal(in), ((HMIBean) bean).writeExternal(out) 이렇게 처리되도록 변경함 (bean이 기본적으로 JComponent로 선언되어 있어서 JComponent의 serialVersionUID가 같이 저장되어 JRE버전 간 호환성 문제가 발생되어 수정함) 2) beanClassName을 bean보다 먼저 저장하도록 하여, read시에 bean객체를 생성할 수 있도록 순서를 변경함.
  1. serialVersionUID = 1, externalVersion = 2로 변경됨 Java Serialize 과정에서 serialVersionUID가 이미 검사되므로 이 버전을 변경하지 않고, 내부적으로 처리가 변경된 부분을 대비하여 externalVersion을 운영한다. externalVersion의 기존 버전은 1이므로 위에서 변경된 사항은 버전 2이상일 경우에만 적용하며 읽혀진 버전이 1일 때에는 기존 읽기 방식으로 진행되어야 한다.
