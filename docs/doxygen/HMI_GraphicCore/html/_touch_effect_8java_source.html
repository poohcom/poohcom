1420  // 2010. 1. 6 수정
1421  // AWTEventQueue 스레드에 의해 수행될 경우 openWindow()시 대기창이 표시되지 않으므로 openWindow()
1422  // 부분만을 스레드 처리하여 실행하였다.

1424  // 2010. 1. 27 수정
1425  // 화면 열기 및 동작 실행을 별도의 스레드에 의해 실행될 때, 선후 관계에 의하여

1437  }
1438  }

1456  }

1461  }
