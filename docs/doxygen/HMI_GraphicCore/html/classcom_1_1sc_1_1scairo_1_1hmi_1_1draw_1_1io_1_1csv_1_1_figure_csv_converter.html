Detailed Description

FigureCsvConverter 화면 객체를 CSV에 정의된 Token에 따라 속성별로 컴마분리 문자열로 만들거나, 읽어들이는데 사용되는 유틸리티 클래스.

수정일자: 2014. 3. 21 수정자: BIGGI 수정내용:

  1. Bean 클래스 javolution XML 쓰기/읽기 시에 문제(Bar)가 발생되어 기존 코드(serialize)를 살리고 XML 저장 코드를 주석 처리함. load(HMIFigure, int, String[]), save(HMIFigure)
