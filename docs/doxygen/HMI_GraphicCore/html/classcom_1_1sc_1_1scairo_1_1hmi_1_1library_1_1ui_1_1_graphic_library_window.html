Detailed Description

시스템 편집기 내의 화면 편집기에서는 기본적으로 제공된 SVG, 컴포넌트, 이미지 등의 기본 심볼 등을 이용하여 화면을 구성하실 수 있도록 하는 라이브러리 창.
화면 왼쪽에는 JTree 형태로 GraphicLibrary 정보를 표시하며, 화면 오른쪽은 Category 별 Symbol 정보를 표시한다.
setTool(BeanCreationTool, SVGSymbolCreationTool, ImageSymbolCreationTool, GroupSymbolCreationTool, Tool) 메서드를 이용하여, 라이브러리 창(GraphicLibraryWindow)에서 선택된 심볼을 화면 편집기에 설정하기 위해 사용되는 Tool 정보를 필히 설정해야한다.

JTree 에서 우클릭시 나타나는 팝업 메뉴를 이용하여, 추가적으로 사용자가 제작한 SVG, 이미지, 컴포넌트를 라이브러리에 등록하여 사용할 수 있다.

수정일시: 2014. 3. 20 수정자: BIGGI 수정내용: 1. drawSymbol()에서 Symbol에서 얻은 데이터를 beanTool에 주입할 때, 심볼의 유형이 ILibraryConstants#TYPE_BEAN일 경우 Bean 클래스 명을 주입하도록 변경함. 이는 GraphicLibrary 저장시 컴포넌트 serialize의 경우 JRE버전 문제 때문에 수정되었음.
